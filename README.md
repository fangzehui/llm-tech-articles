# llm-tech-articles

半自动数字人社区配套技术文章库 —— 作者 **方泽辉**。

本仓库收录"LLM 工程化与运维"主题的系列长文，每一篇都尝试把生产环境踩过的坑、踩坑后形成的工程取舍、以及对应的可复用代码沉淀下来。文章首发于 CSDN，源文件以中文 Markdown 形式同步在这里，便于读者按需检索、对照代码、提交勘误。
