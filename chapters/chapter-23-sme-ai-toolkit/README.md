# Chapter 23 - 中小微企业 AI 落地成本测算器

本目录是文章《[23 中小微企业低成本用上 AI 的实操指南](../../23-中小微企业低成本用上AI的实操指南.md)》的配套示例代码。

## 项目介绍

把"中小微企业要不要上 AI"这件事变成一道可算的数学题：从三大优先级场景（营销获客 / 内部效率 / 核心业务）出发，给出**月度成本 vs 年度收益 vs 回收期**的最小可读估算器，并支持自定义企业人数 / 营收。

- **`main.py`**：定义 `AIToolPlan`、`estimate_savings()`、`payback_months()`，覆盖 6 个可立刻上手的工具组合（AI 写文 / AI 客服 / AI 办公 / AI 财务 / AI 合同审查 / 行业 SaaS）；
- **`tests/test_main.py`**：3 个 pytest 用例覆盖成本累加、年化收益和回收期边界。

## 安装步骤

```bash
cd chapters/chapter-23-sme-ai-toolkit
pip install -r requirements.txt
python main.py
pytest tests/ -v
```
