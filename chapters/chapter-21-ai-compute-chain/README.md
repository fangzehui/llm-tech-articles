# Chapter 21 - AI 算力产业链投资标的打分器

本目录是文章《[21 美光财报背后的 AI 算力产业链投资机会](../../21-美光财报背后的AI算力产业链投资机会.md)》的配套示例代码。

## 项目介绍

把"美光 FY26Q3 数据中心营收 +200% YoY"这条信号往上下游延伸三个环节（上游核心芯片 / 中游算力调度 / 下游场景落地），用一个最小可读的 Python 脚本演示"产业链标的打分 + 仓位分配"的工程做法：

- **`main.py`**：定义 `Stock` / `score()` / `allocate()` 三个最小可读单元，覆盖三大主线的代表标的，按"成长 × 估值 × 行业地位"加权打分，给出建议仓位百分比；
- **`tests/test_main.py`**：3 个 pytest 用例覆盖打分一致性、仓位归一化和分层排序。

> ⚠️ 数据为教学示例（非实时行情），不构成任何投资建议。

## 安装步骤

```bash
cd chapters/chapter-21-ai-compute-chain
pip install -r requirements.txt   # 仅 pytest
python main.py                    # 看输出表
pytest tests/ -v                  # 跑测试
```
