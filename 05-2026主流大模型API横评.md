## 维度一：推理性能评测

### 评测方法

我们设计了5个核心测试场景，每个场景使用统一Prompt进行评测：

1. **代码生成**：Python/JavaScript各50题
2. **中文对话**：日常问答、写作、翻译
3. **长上下文**：10万字文档摘要与分析
4. **数学推理**：高考数学、竞赛题各50题
5. **多模态理解**：图文混合理解50题

评测指标：
- **准确率**：客观题正确率
- **响应时间**：P50/P95/P99延迟
- **上下文保持**：长对话中的一致性

### 场景化成本优化策略

```python
class CostOptimizationStrategy:
    """
    成本优化策略
    
    核心思路：根据任务难度分级，用合适的模型处理对应的任务
    """
    
    # 任务分级
    TASK_TIERS = {
        "tier1_high_value": {
            "examples": ["合同审查", "代码审计", "专业翻译"],
            "recommended_models": ["Claude Sonnet 4.6", "GPT-5.4"],
            "priority": "quality"
        },
        "tier2_standard": {
            "examples": ["内容创作", "问答对话", "报告生成"],
            "recommended_models": ["GPT-5.4 Mini", "Claude 4 Haiku", "Qwen3.5 72B"],
            "priority": "balanced"
        },
        "tier3_high_volume": {
            "examples": ["评论分类", "标签提取", "日志分析"],
            "recommended_models": ["Qwen3.5 7B", "Gemini 3 Flash Preview"],
            "priority": "cost"
        }
    }
    
    @staticmethod
    def estimate_savings(
        tier_distribution: dict,
        baseline_model: str = "GPT-5.4"
    ) -> dict:
        """
        估算成本节省
        
        假设当前全部使用GPT-5.4，优化后的成本对比
        """
        # 各模型的相对成本（以GPT-5.4为基准1.0）
        relative_costs = {
            "GPT-5.4": 1.0,
            "Claude Sonnet 4.6": 1.3,
            "GPT-5.4 Mini": 0.04,
            "Claude 4 Haiku": 0.06,
            "Gemini 3 Flash Preview": 0.03,
            "Qwen3.5 72B": 0.008,
            "Qwen3.5 7B": 0.002,
        }
        
        baseline_cost = sum(
            ratio * relative_costs[baseline_model]
            for ratio in tier_distribution.values()
        )
        
        optimized_cost = sum(
            ratio * relative_costs[str(model).split("'")[1]]
            for tier_name, ratio in tier_distribution.items()
            for model in [CostOptimizationStrategy.TASK_TIERS[tier_name.replace("tier", "tier")]["recommended_models"][0]]
        )
        
        return {
            "baseline_cost": baseline_cost,
            "optimized_cost": optimized_cost,
            "savings_percent": (1 - optimized_cost/baseline_cost) * 100
        }
```
